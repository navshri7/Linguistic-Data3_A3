## Repository Structure
* `code.py`: Python script for feature extraction, part-of-speech tagging, and calculating dependency distances.
* `code.r`: R scripts for statistical modeling and Principal Component Analysis (PCA) visualizations.

## Methodology

1. **Preprocessing:** Tokenization and morphological analysis of Hindi text.
2. **Feature Extraction:** Calculating Mean Dependency Distance (MDD) and word frequency.
3. **Statistical Analysis:** * Running **PCA** to reduce the dimensionality of complexity metrics.
* Plotting clusters to see how different Hindi registers (e.g., formal vs. informal) behave.
4. **Hypothesis Testing:** Using t-tests to validate the relationship between complexity and duration.

* **Python Libraries:** `pandas`, `numpy`, `Stanza` (for dependency parsing), `scikit-learn`.
* **R Libraries:** `ggplot2`, `factoextra` (for PCA visualization), `lme4`.
